#### 2. 알고리즘 문제 풀이
- BOJ 에서 그 주의 강의 내용을 응용할 수 있는 문제 풀기
  - 주차별 링크해둔 문제집의 <b>가장 위에 있는 기본 문제</b>는 필수
  - 이 외의 문제들은 자율적으로 선택해서 풀면 됨
  - 푼 문제는 현재 레포지토리 내 본인 폴더에 올리기 
- 잔디심기 챌린지(1일1문제풀기) 진행

####  3. 온라인 스터디
- 1주일에 한번, 온라인으로 모여서 어려웠던 내용을 질문하거나 특이한 풀이를 공유하는 시간을 갖기로 함
